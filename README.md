# Network Analysis Mx

Herramienta de análisis de red para **Linux y Windows** con 7 módulos integrados.

## 🛠️ Herramientas incluidas

| # | Herramienta          | Función                                                                 |
|---|----------------------|-------------------------------------------------------------------------|
| 1 | 📡 Ping Tool          | Enviar paquetes ICMP a un host                                          |
| 2 | 🔬 Advanced Banner    | Obtener banner completo de servicios                                    |
| 3 | 🎯 Banner Grabbing    | Extraer información de servicios HTTP, FTP, SSH, etc.                   |
| 4 | 🔌 MAC Lookup         | Buscar fabricante por dirección MAC                                     |
| 5 | 📍 IP Geolocation     | Localizar IP en mapa (ciudad, país, coordenadas)                        |
| 6 | 🔒 SSL Checker        | Validar certificados SSL/TLS de dominios                                |
| 7 | 🔄 Port Checker       | Escanear puertos abiertos en un objetivo                                |
